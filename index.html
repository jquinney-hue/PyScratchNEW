Documentation

Every sprite's code starts with:
from pyscratch import *

Each sprite gets its own Python file. Multiple def functions can run simultaneously — game_start, on_click, and on_keypress all run in parallel.
Motion
move_steps(steps)

Moves forward in current direction.

turn(degrees)

Rotates clockwise.

go_to(x, y) or go_to("random")

Teleports sprite.

glide_to(x, y, secs)

Smoothly moves over time.

point_towards(x, y)

Aims at coords, sprite name, "mouse pointer", or a degree number.

set_x(n) / set_y(n) / change_x(n) / change_y(n)
get_x() / get_y() / get_direction()
on_edge() & bounce()

Call on_edge() first, then bounce() to reflect direction.

Looks
say(message, secs=2)

Speech bubble, pauses execution.

set_costume(name) / next_costume()
set_stage(name) / next_stage()
set_size(%) / change_size(n) / show() / hide()
Events (define these functions)
def game_start():

Runs when ▶ is pressed.

def on_click():

Runs when this sprite is clicked.

def on_keypress(key):

key is "space", "up", "a", etc.

broadcast(msg) / def broadcast_receive(msg):
Control
wait(secs)

Pauses this function (other functions keep running).

stop()

Stops everything.

while True: / if / for

Standard Python — all work natively.

Sensing
touching("SpriteName") or touching("mouse pointer")
distance_to("SpriteName")
key_pressed("space")

True while key is held.

ask("question")

Returns user's answer as a string.

Variables & Random
display_variable("score", score)

Shows a variable monitor on stage.

hide_variable("score")
import random

Standard Python random works: random.randint(1, 10)

# Example: bouncing ball
from pyscratch import *
import random

def game_start():
    set_size(50)
    go_to(0, 0)
    while True:
        move_steps(4)
        if on_edge():
            bounce()
        wait(0.016)

def on_click():
    go_to("random")
